# AXSI_Shell

Масштабируемая оболочка для Hyprland на базе Quickshell (v0.2.1+).

## Структура проекта

Проект организован по модульному принципу для легкого расширения:

- `shell.qml`: Основной файл конфигурации.
- `components/`: Содержит определения окон (например, `Bar.qml`).
- `widgets/`: Содержит переиспользуемые виджеты.
- `services/`: Содержит синглтоны для работы с данными (время, системная информация).

## Запуск

Для тестирования запустите команду в директории проекта:

```bash
quickshell
```

## Автозапуск в Hyprland

Добавьте в `~/.config/hypr/hyprland.conf`:

```
exec-once = quickshell -c ~/.config/quickshell/AXSI_Shell/shell.qml
```
*(Замените путь на актуальный)*

## Реализованные модули

1. **Workspaces**: Индикатор рабочих столов с подсветкой активного и возможностью переключения кликом.
2. **Clock**: Форматированные часы.
3. **KbdLayout**: Отображение текущей раскладки клавиатуры (US, RU и т.д.).
4. **Audio (Pipewire)**: Отображение уровня громкости. Клик открывает меню выбора устройства вывода (Sink).
5. **SysTray**: Полноценный системный трей с поддержкой иконок, подсказок и контекстных меню.
